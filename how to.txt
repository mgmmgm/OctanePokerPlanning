Prerequisites:
--------------------
NodeJS
MongoDB


Modules Installation: (npm install)
-------------------
express
mongoose
body-parser

How to install:
-------------------
run 'npm install' where 'package.json' file exist (in root folder)


To start:
-------------------
node app.js (to start the node server)


how to run MongoDB:
-----------------------
in folder: C:\Program Files\MongoDB\Server\3.2\bin\
run the command: mongod --dbpath=C:/Mongo
